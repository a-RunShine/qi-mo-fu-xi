## 二、`Arrays`(数组工具)★

### 2.1 完整 API 速查

| 方法 | 说明 | C++ 对照 |
|---|---|---|
| `static void sort(int[] a)` | 基础类型数组升序 | `std::sort` |
| `static <T> void sort(T[] a, Comparator<? super T> c)` | 对象数组,可自定义比较器 | `std::sort` + lambda |
| `static int binarySearch(int[] a, int key)` | 二分查找(数组必须**有序**) | `std::binary_search` |
| `static void fill(int[] a, int val)` | 填充 | `std::fill` |
| `static String toString(int[] a)` | 转字符串(debug 用) | 无内建 |
| `static <T> List<T> asList(T... a)` | 转 List(**定长!**) | `std::vector` 一次性构造 |
| `static IntStream stream(int[] a)` | 转 IntStream(Java 8+) | 无 |
| `static int[] copyOf(int[] a, int newLength)` | 拷贝(可扩/缩) | `std::vector` 拷贝构造 |
| `static boolean equals(int[] a, int[] a2)` | 数组内容比较 | `std::equal` |
| `static void setAll(int[] a, IntUnaryOperator gen)` | 函数式填充(Java 8+) | `std::generate` |

### 2.2 关键差异(必看)

> **重点**:`Arrays.sort(int[])`(基础类型) 和 `Arrays.sort(Integer[])`(对象) 是**两个不同重载**。基础类型**不能**传 `Comparator`(编译器不让你写 `(a, b) -> b - a`)。
>
> **解决方法 1**:`Arrays.sort` 配对象类型:`Arrays.sort(Integer[], (a, b) -> b - a)`。
> **解决方法 2**:转成 `List` 排序,见下。

### 2.3 最小代码:排序 + 自定义比较器 + 二分 + 填充

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // 1. 基础类型升序
        int[] a = {3, 1, 4, 1, 5, 9, 2, 6};
        Arrays.sort(a);
        System.out.println(Arrays.toString(a));  // [1, 1, 2, 3, 4, 5, 6, 9]

        // 2. 基础类型降序?不能直接 sort!两种解法
        // 解法 A:包装成 Integer[] 再 sort + Comparator
        Integer[] a2 = {3, 1, 4, 1, 5, 9, 2, 6};
        Arrays.sort(a2, (x, y) -> y - x);  // 降序
        System.out.println(Arrays.toString(a2));  // [9, 6, 5, 4, 3, 2, 1, 1]

        // 解法 B:转 List 用 Collections.sort(见下节)
        // 3. 二维数组按第二个字段升序(LeetCode 56 合并区间简化)
        int[][] points = {{1, 3}, {2, 2}, {3, 1}};
        Arrays.sort(points, (p, q) -> p[1] - q[1]);  // 按 [1] 升序
        // 输出:[[3, 1], [2, 2], [1, 3]]

        // 4. 二分查找(必须先排序!)
        int[] sorted = {1, 2, 3, 4, 5, 6, 7, 8, 9};
        int idx = Arrays.binarySearch(sorted, 5);  // 返回 4(index)
        int miss = Arrays.binarySearch(sorted, 10); // 返回 -10(负数表示未找到)

        // 5. 填充
        int[] fillArr = new int[5];
        Arrays.fill(fillArr, 7);
        System.out.println(Arrays.toString(fillArr));  // [7, 7, 7, 7, 7]

        // 6. 数组比较
        int[] x = {1, 2, 3}, y = {1, 2, 3};
        System.out.println(Arrays.equals(x, y));  // true(不是 == !)
    }
}
```

## 三、`Collections`(集合工具)

### 3.1 完整 API 速查

| 方法 | 说明 | C++ 对照 |
|---|---|---|
| `static <T> void sort(List<T> list)` | 列表升序(元素必须实现 Comparable) | `std::sort` |
| `static <T> void sort(List<T> list, Comparator<? super T> c)` | 自定义比较器排序 | `std::sort` + lambda |
| `static void reverse(List<?> list)` | 原地反转 | `std::reverse` |
| `static T max(Collection<? extends T> coll)` | 最大值 | `std::max_element` |
| `static T min(Collection<? extends T> coll)` | 最小值 | `std::min_element` |
| `static int frequency(Collection<?> c, Object o)` | 出现次数 | `std::count` |
| `static void swap(List<?> list, int i, int j)` | 交换两个位置 | `std::swap` |
| `static int binarySearch(List<? extends Comparable<? super T>> list, T key)` | 二分(列表必须有序) | `std::binary_search` |

| `static boolean disjoint(Collection<?> c1, Collection<?> c2)` | 是否不相交(无共同元素) | 无内建,需手写 |

### 3.2 最小代码:list 排序 + 反转 + 最值

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        List<Integer> list = new ArrayList<>(Arrays.asList(3, 1, 4, 1, 5, 9, 2, 6));

        // 1. 排序
        Collections.sort(list);                  // 升序:[1, 1, 2, 3, 4, 5, 6, 9]
        Collections.sort(list, (a, b) -> b - a);  // 降序:[9, 6, 5, 4, 3, 2, 1, 1]

        // 2. 反转
        Collections.reverse(list);  // 翻转列表

        // 3. 最大 / 最小(返回元素,不是下标)
        int maxVal = Collections.max(list);  // 9
        int minVal = Collections.min(list);  // 1

        // 4. 频次
        int cnt = Collections.frequency(list, 1);  // 1 出现几次

        // 5. 交换
        Collections.swap(list, 0, 1);  // 交换第 0 和第 1 个

        // 6. 批量添加
        List<Integer> list2 = new ArrayList<>();
        Collections.addAll(list2, 1, 2, 3, 4, 5);  // 等价于 add 5 次

        // 7. 不相交判断
        boolean noCommon = Collections.disjoint(
            Arrays.asList(1, 2, 3),
            Arrays.asList(4, 5, 6)
        );  // true
    }
}
```

### 3.3 C++ 对照

```cpp

```

### 3.4 关键差异(★ Java 8 后用 `List.sort` 优先)

> **重要**:`Collections.sort(list)` 和 `list.sort(comparator)` 功能等价。但 `List.sort` 是 Java 8 新增的 `List` 接口默认方法,**写法更简洁、稍快一点**。`Collections.sort` 是 Java 8 之前的"老写法",现已不推荐。
>
> **C++ 选手注意**:C++ 是 `sort(v.begin(), v.end())`(算法 + 迭代器),Java 是 `list.sort(c)`(方法是 List 自己的)。**调用对象不同,效果一样**。

| 写法 | 推荐度 | 备注 |
|---|---|---|
| `list.sort((a, b) -> a - b)` | ★★★ | Java 8+,默认写法 |
| `Collections.sort(list)` | ★★ | 老写法,仅出现在早期题解 |
| `Collections.sort(list, (a, b) -> a - b)` | ★★ | 老写法 + 自定义 |

详见本章第九节 `List.sort`。

### 4.2 最小代码:距离公式 + 浮点比较

```java
public class Main {
    public static void main(String[] args) {
        // 1. 基础运算
        int a = Math.max(3, 5);       // 5
        int b = Math.abs(-7);          // 7
        double c = Math.sqrt(16);      // 4.0
        double d = Math.pow(2, 10);    // 1024.0
        double pi = Math.PI;           // 3.14159...

        // 2. 三角函数(弧度!)
        double rad = Math.toRadians(30);  // 30 度 → 弧度
        double s = Math.sin(rad);          // 0.5

        // 3. 随机数
        double r = Math.random();  // [0, 1)
        int dice = (int) (Math.random() * 6) + 1;  // [1, 6]

        // 4. 浮点比较(★ 浮点精度坑)
        double x = 0.1 + 0.2;
        System.out.println(x);              // 0.30000000000000004(不是 0.3!)
        System.out.println(x == 0.3);       // false(直接 == 永远 false)
        System.out.println(Math.abs(x - 0.3) < 1e-9);  // true(eps 比较)

        // 5. 实际应用:两点欧氏距离
        double dist = Math.sqrt(Math.pow(3 - 0, 2) + Math.pow(4 - 0, 2));  // 5.0
    }
}
```

### 4.4 ★★ 坑点:浮点精度,不要直接 `==`(详见 06 坑 7)

```java
// 错误:浮点数永远不要直接 ==

if (a == 0.3) { /* 永远不会进 */ }

// 正确:eps 比较

if (Math.abs(a - 0.3) < EPS) { /* 相等 */ }
```

> **原因**:IEEE 754 浮点数无法精确表示 0.1、0.2、0.3,所有浮点运算都有微小误差。**所有浮点比较都要加 eps**(epsilon,1e-9 是常用阈值)。

## 五、`String`(字符串,不可变)★★★

> **Java 不可变**:`String` 一旦创建就不能改。`s = s + "x"` 实际上是创建了一个**新 String 对象**,把引用赋给 `s`。要"原地改"用 `StringBuilder`(见下节)。
>
> **C++ 选手视角**:C++ 的 `std::string` 是可变的,`s += "x"` 是真改原对象。Java 的 `String` 概念上更接近 `const std::string`,或者 `char* p = "abc"`(只读指针)。**这是 C++ 转 Java 第一个概念落差,务必记牢**。

### 5.1 完整 API 速查

| 方法 | 说明 | C++ 对照 |
|---|---|---|
| `char charAt(int index)` | 取 index 字符 | `s[i]` |
| `int indexOf(String str)` / `lastIndexOf` | 找子串位置(返回 -1 表示没有) | `s.find(...)` |
| `String substring(int begin)` / `substring(int begin, int end)` | 子串(包头不包尾) | `s.substr(...)` |

### 5.4 ★★★ 坑点 1:`==` vs `equals`(C++ 选手第 1 大坑,详见 06 坑 11)

> **C++ 选手的致命错**:在 C++ 里 `s == t` 是**内容比较**。Java 里 `==` 是**引用比较**(比地址),`equals` 才是**内容比较**。**C++ 选手转 Java 几乎必踩**。

```java
// 错误:== 比的是引用,不是内容
String s1 = "abc";
String s2 = new String("abc");
System.out.println(s1 == s2);      // false!(比的是地址)
System.out.println(s1.equals(s2)); // true(内容比较)

// 更隐蔽的坑:字符串常量池
String a = "hello";
String b = "hello";
System.out.println(a == b);        // true(都指向常量池同一对象)
String c = new String("hello");
System.out.println(a == c);        // false(a 是常量池,c 是 new 出来的)
```

```cpp
// C++ 没这个问题,s == t 直接比内容
string s1 = "abc";
string s2 = "abc";
cout << (s1 == s2);  // true,无误
```

> **金句**:**"在 Java 里,字符串比内容永远用 `.equals()`,`==` 只用于比 null 或比对象地址"**。这条规则适用于 `String` / `Integer` / `Long` / `Double` / 任何包装类。

### 5.5 ★★★ 坑点 2:`length` vs `length()` vs `size()`(详见 06 坑 12)

> **C++ 选手困惑**:C++ 全靠 `s.size()` / `v.size()`。Java 分三套:**数组用 `length`(无括号),字符串用 `length()`(有括号),集合用 `size()`(有括号)**。第一次写会卡。

```java
int[] arr = {1, 2, 3};
int a = arr.length;             // 数组:字段,无括号
int b = "abc".length();         // 字符串:方法,有括号
int c = list.size();            // List/Set/Map:方法,有括号
int d = map.size();             // Map 也有 size()
```

> **记忆口诀**:**数组 length 字段不挂括号,字符串 length 挂括号,集合 size 挂括号**。C++ 全是 `.size()` 一统天下,Java 因为历史原因分成 3 套。

## 六、`StringBuilder`(可变字符串,刷题天天用)★

> **概念铺垫**:因为 `String` 不可变,所有"原地改字符串"的需求(拼接、删除、反转)都用 `StringBuilder`。刷题时 80% 的"我要拼一个长字符串"场景,都用 `StringBuilder`。

### 6.2 最小代码:数字反转(LeetCode 7)+ 字符串拼接

```java
public class Main {
    public static void main(String[] args) {
        // 1. 数字反转(LeetCode 7 整数反转 简化版)
        int x = 12345;
        StringBuilder sb = new StringBuilder(String.valueOf(x));  // "12345"
        sb.reverse();                                             // "54321"
        int result = Integer.parseInt(sb.toString());             // 54321

        // 2. 字符串拼接(★ 刷题天天用)
        // 错误:N 个字符串 + 拼接 = O(n²) 慢!
        String s = "";
        for (int i = 0; i < 100000; i++) {
            s += "x";  // 每次都创建新 String 对象,慢
        }
        // 正确:StringBuilder 拼接 = O(n) 快
```

```java
// 思路:大数乘法用 StringBuilder 倒着累加,避免 BigInteger 的开销
public String multiply(String num1, String num2) {
    if (num1.equals("0") || num2.equals("0")) return "0";
    int m = num1.length(), n = num2.length();
    int[] pos = new int[m + n];  // 每一位的累加和
    for (int i = m - 1; i >= 0; i--) {
        for (int j = n - 1; j >= 0; j--) {
            int mul = (num1.charAt(i) - '0') * (num2.charAt(j) - '0');
            int p1 = i + j, p2 = i + j + 1;
            int sum = mul + pos[p2];
            pos[p1] += sum / 10;
            pos[p2] = sum % 10;
        }
    }
    StringBuilder sb = new StringBuilder();
    for (int p : pos) {
        if (!(sb.length() == 0 && p == 0)) sb.append(p);
    }
    return sb.length() == 0 ? "0" : sb.toString();
}
```

### 7.5 实战金句

### 8.4 写法 4:链式调用(★ 多键排序神器)

> **第一次见的概念铺垫**:对比单字段排序(`(a, b) -> a[0] - b[0]`),链式调用是**多键排序**——先按一个字段排,字段相同再按下一个字段排。**LeetCode 406 / 179 / 56 等多键排序题必备**。

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // 1. 多键排序:按第二个字段升序,再按第一个字段升序
        int[][] points = {{1, 3}, {2, 2}, {3, 1}, {1, 2}};
        Arrays.sort(points, Comparator
            .comparingInt((int[] p) -> p[1])    // 主键:第二个字段
            .thenComparingInt(p -> p[0]));      // 次键:第一个字段
        // 排完:[[1, 2], [3, 1], [2, 2], [1, 3]]

        // 2. 字符串 + 数字混合排序
        String[] words = {"apple", "banana", "cherry"};
        Arrays.sort(words, Comparator
            .comparingInt(String::length)        // 按长度升序
            .thenComparing(String::compareTo));   // 长度相同按字典序

        // 3. 降序链式
        Arrays.sort(points, Comparator
            .comparingInt((int[] p) -> p[1])
            .reversed());  // 整体反转 → 第二个字段降序

        // 4. 链式 + 然后升序(降序降序)
        // LeetCode 406 标准:先 h 降序,再 k 升序
        int[][] people = {{7, 0}, {4, 4}, {7, 1}, {5, 0}, {6, 1}, {5, 2}};
        Arrays.sort(people, Comparator
            .comparingInt((int[] p) -> p[0]).reversed()  // h 降序
            .thenComparingInt(p -> p[1]));               // k 升序
    }
}
```

**关键 API**:
- `Comparator.comparingInt(ToIntFunction)` - 按 int 字段升序
- `Comparator.comparingLong(ToLongFunction)` - 按 long 字段升序
- `Comparator.comparing(Function)` - 按对象字段升序
- `.thenComparingInt(...)` - 第二键(int)
- `.thenComparingLong(...)` - 第二键(long)
- `.thenComparing(...)` - 第二键(对象)
- `.reversed()` - 整体反转(降序)

**适用场景**:**多键排序必用**,比手写 lambda `((a, b) -> a[0] != b[0] ? a[0] - b[0] : a[1] - b[1])` 简洁 5 倍。

### 8.5 写法 5:`Collections.reverseOrder()`(最简降序)

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Integer[] arr = {3, 1, 4, 1, 5, 9, 2, 6};
        Arrays.sort(arr, Collections.reverseOrder());  // 降序,一行搞定

        // 也可以用 Comparator.reverseOrder()(Java 8+)
        Arrays.sort(arr, Comparator.reverseOrder());
    }
}
```

**适用场景**:**单字段降序**用这个最简洁,比 `(a, b) -> b - a` 还能防止**整数溢出**(虽然一般不会)。

### 8.6 记忆口诀(★ 必背)

| 场景 | 写法 | 备注 |
|---|---|---|
| 升序(单字段) | `(a, b) -> a - b` | 最常用 |
| 降序(单字段) | `(a, b) -> b - a` 或 `Collections.reverseOrder()` | 后者更安全 |
| 升序(多字段) | `Comparator.comparingInt(...).thenComparingInt(...)` | **多键神器** |
| 降序(多字段) | `.comparingInt(...).reversed().thenComparingInt(...)` | 整体反转 |
| 字符串字段 | `Comparator.comparing(String::length)` | 方法引用 |

> **实战金句**:**"单字段 lambda,多字段链式,降序优先 reverseOrder"**。

### 9.1 `Map.getOrDefault(key, defaultValue)`(★★★ LeetCode 1/347 天天用)

> **C++ 选手视角**:C++ 用 `m.count(k) ? m[k] : default;` 或 `m.find(k) != m.end() ? m[k] : default;`(2-3 行)。Java 一行搞定,这就是 Java 8 的甜头。

**场景**:统计字符/元素出现次数,key 不存在时给个默认值。

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // 1. 统计频次(最常用)
        Map<Character, Integer> freq = new HashMap<>();
        String s = "abracadabra";
        for (char c : s.toCharArray()) {
            freq.put(c, freq.getOrDefault(c, 0) + 1);
        }
        System.out.println(freq);  // {a=5, b=2, r=2, c=1, d=1}

        // 2. LeetCode 387 字符串中第一个唯一字符(完整版见 class Solution)
        // 思路:统计每个字符出现次数,再扫一遍找第一个 count == 1 的
        int idx = new Solution().firstUniqChar("abracadabra");  // 调用,返回 4('c' 是第一个唯一字符)
        System.out.println(idx);

        // 3. C++ 对照
        // unordered_map<char, int> freq;
        // for (char c : s) freq[c]++;
        // // C++ 的 unordered_map operator[] 会自动默认初始化为 0,不需要 getOrDefault
    }
}

// LeetCode 模式可提交版
class Solution {
    public int firstUniqChar(String s) {
        Map<Character, Integer> count = new HashMap<>();
        for (char c : s.toCharArray()) {
            count.put(c, count.getOrDefault(c, 0) + 1);
        }
        for (int i = 0; i < s.length(); i++) {
            if (count.get(s.charAt(i)) == 1) return i;
        }
        return -1;
    }
}
```

> **金句**:**"统计频次 = 循环 + getOrDefault + 1"**,是 LeetCode 哈希题第一模板。

### 9.2 `Map.computeIfAbsent(key, k -> new ArrayList<>())`(★★★ BFS/图遍历建邻接表)

**场景**:邻接表(BFS/DFS)、按 key 分组。

```java
import java.util.*;
```

### 9.3 `List.sort(Comparator)`(Java 8 新增,`Collections.sort` 替代)

> **C++ 选手视角**:C++ 是 `std::sort(v.begin(), v.end())`(算法 + 迭代器)。Java 是 `list.sort(c)`(方法是 List 自己的)。调用对象不同,效果一样。

**关键差异**:

| 维度 | `Collections.sort(list)` | `List.sort(comparator)` |
|---|---|---|
| 引入版本 | Java 2 | **Java 8** |
| 调用方式 | 工具类静态方法 | List 接口**默认方法** |
| 可读性 | 老式 | **新式,推荐** |
| 性能 | 略慢 | **略快**(内部走 `Arrays.sort`) |

```java
import java.util.*;
```

### 9.5 `String.chars()` / `String.codePoints()`(★ 扫一眼即可)

```java
public class Main {

        // 2. codePoints():返回完整 Unicode 码点 IntStream(处理中文/emoji 更准)
        s.codePoints().forEach(c -> System.out.println((char) c));

        // 3. 推荐替代:toCharArray() 更直观
        for (char c : s.toCharArray()) {
            System.out.println(c);
        }
    }
}
```

> **金句**:**"看到 `chars()` / `codePoints()` 知道是流式处理,实际用 `toCharArray` 代替"**。要详细学流式处理去看官方文档,速成阶段不深究。

## 十、`Stream` API 速查(低优先级,扫一眼即可)

> **范围**:本节**只列最常用 4-5 个**,Java 8 写法。Stream 在刷题里**不推荐**作为主力(性能略差),但作为辅助(转换、过滤)很方便。
>
> **金句**:**"Stream 写法更函数式,但性能略差,刷题优先用 for 循环 + 工具类,Stream 用于辅助"**。

### 10.1 4 个常用写法

```java
import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        List<Integer> list = Arrays.asList(3, 1, 4, 1, 5, 9, 2, 6);

        // 1. filter + collect(替代手写过滤循环)
        List<Integer> evens = list.stream()
            .filter(x -> x % 2 == 0)
            .collect(Collectors.toList());  // [4, 2, 6]

        // 2. mapToInt.sum(替代 accumulate 求和)
        int sum = list.stream()
            .mapToInt(Integer::intValue)  // Integer → int
            .sum();  // 30
        // C++:int sum = accumulate(list.begin(), list.end(), 0);

        // 3. distinct(替代 unique 去重)
        List<Integer> unique = list.stream()
            .distinct()
            .collect(Collectors.toList());  // [3, 1, 4, 5, 9, 2, 6]
        // C++:vector<int> unique(list.begin(), list.end());
        //      unique(unique.begin(), unique.end());

        // 4. sorted(替代 sort)
        List<Integer> sorted = list.stream()
            .sorted()
            .collect(Collectors.toList());  // [1, 1, 2, 3, 4, 5, 6, 9]
        // C++:sort(list.begin(), list.end());

        // 5. 数组转 Stream(注意:基本类型数组要特殊处理)
        int[] arr = {1, 2, 3};
        int arrSum = Arrays.stream(arr).sum();      // 推荐:Arrays.stream
        int arrSum2 = IntStream.of(arr).sum();      // 等价:IntStream.of
        // 注意:Stream.of(arr) 错误!会把整个 int[] 当一个元素
    }
}
```

### 10.2 实战建议

| 场景 | 推荐 | Stream 替代 |
|---|---|---|
| 求和 / 累加 | for 循环 | `mapToInt.sum()` |
| 去重 | `HashSet` 重建 | `stream.distinct()` |
| 排序 | `Arrays.sort` / `List.sort` | `stream.sorted()` |
| 过滤 | for 循环 + if | `stream.filter()` |
| 转 List | `new ArrayList<>(collection)` | `collect(Collectors.toList())` |

> **金句**:**"Stream 用于转换和过滤,for 循环用于累计和复杂逻辑"**。刷题主力还是 for 循环 + 工具类,Stream 知道有这玩意就行。
