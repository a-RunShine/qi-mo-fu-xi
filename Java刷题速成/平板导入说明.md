# Java 刷题速成 - 平板导入说明

> 9 份 PDF,共 3.4 MB,适配华为平板 + NoteIn App
> 制作时间:2026-06-23

## 方案 A - 华为云盘(首推,华为生态原生)

1. **Mac 上传**
   - 浏览器打开 `https://cloud.huawei.com`
   - 登录华为账号(平板同账号)
   - "我的云盘" → 新建文件夹 `Java刷题速成`
   - 把 9 份 PDF 拖进文件夹,等上传完

2. **平板打开 NoteIn**
   - 主页右下角"+"
   - 选 "导入 PDF"
   - 进入文档列表后,看左上角菜单(三个点或汉堡菜单)
   - 切到"华为云盘"路径
   - 找到 `Java刷题速成/` 文件夹
   - **长按 9 份全选** → 右上角"选择" → 一次性导入

## 方案 B - USB 数据线(最快)

1. USB-C 连 Mac,平板弹出"USB 连接方式"选"**文件传输**"或"**MTP**"
2. Mac Finder 侧栏出现 "HUAWEI MatePad" → 进入 `Internal Storage/Download/`
3. 拖 9 份 PDF 到 Download 文件夹
4. 平板 NoteIn → 导入 PDF → 文档列表切到"文件管理" → 选 Download 路径 → 长按 9 份全选

## 方案 C - 微信文件传输(无数据线/无云盘时)

1. Mac 微信 → "文件传输助手" → 发 `/tmp/Java刷题速成_导入包.zip`(已生成)
2. 平板微信 → 长按 zip → "其他应用打开" → 选 NoteIn
3. NoteIn 提示"导入 1 个 PDF"(zip 不会自动解压,需要平板手动解压一次)

> zip 方案小坑:NoteIn 不会自动解压 zip,需要平板先用"文件管理"解压,再导入
> 备选:Mac 微信不要打包,把 9 份 PDF **多选**后一次性发(微信支持多文件),平板微信长按第一份 → "其他应用打开" → 全选 → NoteIn

## 方案 D - Snapdrop(零安装)

1. Mac 浏览器打开 `https://snapdrop.net`
2. 平板浏览器打开同一个 URL
3. 自动发现对方设备 → 拖 9 份 PDF → 完成

## 在 NoteIn 中的使用建议

- **首次导入后,先打开 00_总览与速成路线.pdf 标"已读"**(建立全局地图)
- **5/6/7 章是重点**(算法模板 + 坑点 + 例题),适合反复标注
- **速查表.pdf** 适合放在第一本书的"常用文档"位置
- **批注颜色建议**:

  - 红色:易错点
  - 绿色:已掌握
