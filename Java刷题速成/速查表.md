## 二、13 容器 API 一行对照(精简版)

| C++ STL | Java | 关键差异 | 常用 API 映射 |
|---|---|---|---|
| `vector<T>` | `ArrayList<T>` | 基本类型用包装 | `push_back→add` / `size→size` / `[]→get/set` |
| `list<T>` | `LinkedList<T>` | 慢,优先 ArrayDeque | `push_back→add` |
| `stack<T>` | **`ArrayDeque<T>`** | 别用 `Stack` | `push→push` / `top→peek` / `pop→pop` |
| `queue<T>` | `ArrayDeque<T>` | 推荐 ArrayDeque | `push→offer` / `front→peek` / `pop→poll` |
| `priority_queue<T>` | `PriorityQueue<T>` | **默认小顶堆!** | 大顶堆:`(a,b)->b-a` 或 `Collections.reverseOrder()` |
| `deque<T>` | `ArrayDeque<T>` | 滑动窗口标配 | `addFirst/Last` / `peekFirst/Last` / `pollFirst/Last` |
| `unordered_map<K,V>` | `HashMap<K,V>` | 允许 null | `[]→get/put` / `find→containsKey` |
| `unordered_set<T>` | `HashSet<T>` | 基于 HashMap | `insert→add` / `find→contains` |
| 无直接对应 | `LinkedHashMap<K,V>` | LRU 缓存 | `new LinkedHashMap<>(cap, 0.75f, true)` |
| `map<K,V>` | `TreeMap<K,V>` | 红黑树 | `lower_bound→lowerKey` / `upper_bound→higherKey` |
| `set<T>` | `TreeSet<T>` | 红黑树 | `lower_bound→lower` |
| `pair<T1,T2>` | `int[]` / `Map.Entry` / 自定义类 | 无内建 pair | |
| `sort` | `Arrays.sort` / `List.sort` | | 稳定:`Arrays.sort(对象)` |

详见:03 容器与 STL 对照。

## 三、8 常用代码片段(直接抄)

### 3.1 排序 + 自定义比较器

```java
Arrays.sort(arr);  // 升序(基本类型)
Arrays.sort(arr, (a, b) -> b - a);  // 降序
Arrays.sort(points, Comparator.comparingInt((int[] p) -> p[1]).thenComparingInt(p -> p[0]));  // 多键
```

### 3.2 二分(找左边界)

```java
int lo = 0, hi = arr.length;
while (lo < hi) {
    int mid = lo + (hi - lo) / 2;
    if (arr[mid] < target) lo = mid + 1;
    else hi = mid;
}
return lo;  // 第一个 ≥ target 的位置
```

### 3.3 双指针(对撞)

```java
int lo = 0, hi = arr.length - 1;
while (lo < hi) {
    int sum = arr[lo] + arr[hi];
    if (sum == target) return new int[]{lo, hi};
    else if (sum < target) lo++;
    else hi--;
}
```

### 3.4 BFS(层序)

```java
ArrayDeque<TreeNode> q = new ArrayDeque<>();
q.offer(root);
while (!q.isEmpty()) {
    int sz = q.size();  // ★ 提前取
    for (int i = 0; i < sz; i++) {
        TreeNode node = q.poll();
        // 业务
        if (node.left != null) q.offer(node.left);
        if (node.right != null) q.offer(node.right);
    }
}
```

### 3.5 DP(一维)

```java
int[] dp = new int[n];
dp[0] = 1; dp[1] = 1;
for (int i = 2; i < n; i++) dp[i] = dp[i-1] + dp[i-2];
return dp[n-1];
```

### 3.6 并查集

```java
int[] parent = new int[n];
for (int i = 0; i < n; i++) parent[i] = i;
int find(int x) { return parent[x] == x ? x : (parent[x] = find(parent[x])); }
void union(int x, int y) { parent[find(x)] = find(y); }
```

### 3.7 单调栈(下一个更大,LeetCode 496/739 风格)

```java
ArrayDeque<Integer> st = new ArrayDeque<>();  // 存下标
int[] res = new int[arr.length];
Arrays.fill(res, -1);
for (int i = 0; i < arr.length; i++) {
    while (!st.isEmpty() && arr[st.peek()] < arr[i]) {
        res[st.pop()] = arr[i];
    }
    st.push(i);
}
```

### 3.8 回溯(三件套)

```java
List<List<Integer>> result = new ArrayList<>();
void backtrack(int[] nums, List<Integer> path, boolean[] used) {
    if (path.size() == nums.length) { result.add(new ArrayList<>(path)); return; }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        path.add(nums[i]); used[i] = true;
        backtrack(nums, path, used);
        path.remove(path.size() - 1); used[i] = false;
    }
}
```
