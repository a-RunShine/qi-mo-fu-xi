## 阅读路线

- **数据库**：从 [`数据库/ai讲解/INDEX.md`](数据库/ai讲解/INDEX.md) 开始（7 章导航 + 跨章主线）→ 按章读 `数据库/ai讲解/第X章_XXX_AI讲解.md` → 用 `数据库/复习题/选择题/` 自测
- **软件工程**：按章读 `软件工程/ai讲解/第X章_XXX_AI讲解.md`，结合 `软件工程/思维导图/` 的 PNG 复习 → 用 `软件工程/复习题/选择题/` 自测
- **Java 刷题速成**：从 [`Java刷题速成/00_总览与速成路线.md`](Java刷题速成/00_总览与速成路线.md) 开始，按 00-07 顺序阅读 → 速查用 [`速查表.md`](Java刷题速成/速查表.md)
- **面向对象原则**：[`面向对象原则/笔记/面向对象原则.md`](面向对象原则/笔记/面向对象原则.md)

## 格式说明

- 所有 `.md` 源文件与配套 `.pdf` 同步提供，PDF 主题色统一为深棕 + 信纸色
- 思维导图为 `.dot`（源）+ `.png`（渲染），可自行用 Graphviz 修改重渲

仅供个人学习使用。
