# 第四章 数据库安全性 — 选择题

1. 某公司数据库中有员工表 Employee。DBA 执行了以下操作：
   ① GRANT SELECT ON Employee TO U1 WITH GRANT OPTION;
   ② U1 将 SELECT 权限授予了 U2；
   ③ DBA 又执行了 GRANT SELECT ON Employee TO U2;
   ④ 随后 DBA 执行 REVOKE SELECT ON Employee FROM U1 CASCADE;
   操作④之后，U2 是否还拥有 SELECT 权限？（  ）
A. 是，因为 U2 从 DBA 直接获得了权限，不受 U1 被回收的影响
B. 否，因为 CASCADE 会级联回收 U1 传播出去的所有权限
C. 是，因为 U2 的权限来自 DBA，CASCADE 只回收 U1 传播的权限
D. 否，因为 U2 的权限依赖于 U1

2. 以下关于 DAC 和 MAC 的说法，正确的是（  ）。
A. DAC 中用户可以自由传播权限，因此比 MAC 更安全
B. MAC 中用户不能改变安全标记，因此比 DAC 更适合需要严格控制的场景
C. DAC 和 MAC 可以同时使用，但 MAC 的优先级低于 DAC
D. MAC 只适用于文件系统，不适用于数据库

3. 以下 SQL 语句中，语法正确的是（  ）。
A. GRANT ALL PRIVILEGES ON TABLE Student TO PUBLIC;
B. GRANT INSERT(sno, sname) ON TABLE Student TO U1;
C. GRANT SELECT ON TABLE Student TO U1 WITH ADMIN OPTION;
D. REVOKE SELECT ON TABLE Student FROM PUBLIC;

4. 某系统中，角色 R1 被授予了 SELECT 权限（WITH ADMIN OPTION），角色 R1 又被授予给用户 U1（WITH ADMIN OPTION）。U1 将角色 R1 授予了 U2。以下说法正确的是（  ）。
A. U2 拥有了角色 R1 中的 SELECT 权限
B. U2 拥有了角色 R1 本身，可以行使 R1 包含的所有权限
C. U1 授予 U2 角色 R1 时，U2 自动获得了 WITH ADMIN OPTION
D. 角色 R1 中的权限发生变化时，U2 的权限不会自动更新

5. 以下关于视图机制在安全性中的作用，说法正确的是（  ）。
A. 视图可以完全替代 GRANT/REVOKE 实现安全控制
B. 通过视图隐藏敏感列，再配合 GRANT 只授予视图权限，可以实现列级安全
C. 视图只能用于查询，不能用于插入和修改
D. 使用 WITH CHECK OPTION 可以防止用户通过视图看到敏感数据

6. 以下关于审计（Audit）的说法，错误的是（  ）。
A. 审计可以记录用户对数据库的所有操作
B. 审计可以在非法操作发生后追查责任人
C. 审计可以实时阻止非法操作的执行
D. 审计会消耗系统资源，影响数据库性能

7. 某军事系统中，数据被标记为"绝密"、"机密"、"秘密"、"公开"四个等级。一个拥有"机密"许可证的用户可以（  ）。
A. 读取"绝密"和"机密"级别的数据
B. 读取"机密"、"秘密"和"公开"级别的数据
C. 读取所有四个级别的数据
D. 只能读取"机密"级别的数据

8. 以下关于 WITH GRANT OPTION 和 WITH ADMIN OPTION 的说法，正确的是（  ）。
A. WITH GRANT OPTION 用于角色授权，WITH ADMIN OPTION 用于对象授权
B. 使用 WITH GRANT OPTION 授权的用户，回收时只能使用 CASCADE
C. WITH ADMIN OPTION 在回收时不受级联影响
D. 两者可以互换使用，没有区别

9. 某 DBA 创建了三个用户：U1（DBA 权限）、U2（RESOURCE 权限）、U3（CONNECT 权限）。以下操作中，哪一项是 U2 无法完成的？（  ）
A. 创建基本表 Student
B. 创建视图 CS_Student
C. 创建新用户 U4
D. 将 Student 表的 SELECT 权限授予 U3

10. 数据加密在数据库安全体系中的定位是（  ）。
A. 替代身份鉴别机制
B. 替代存取控制机制
C. 作为最后一道防线，防止数据被窃取后泄露
D. 主要用于提高查询性能

1.C 2.B 3.A 4.B 5.B 6.C 7.B 8.C 9.C 10.C
