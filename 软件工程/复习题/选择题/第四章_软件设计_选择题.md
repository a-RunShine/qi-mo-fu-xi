7. 模块A以三种方式调用模块B：①传递学号（int类型）；②传递完整的Student结构体，但B只用了学号字段；③传递flag参数，B根据flag值决定执行哪段代码。这三种调用的耦合类型依次是：（  ）

A. 数据耦合、数据耦合、数据耦合
B. 特征耦合、数据耦合、控制耦合
C. 数据耦合、特征耦合、控制耦合
D. 数据耦合、控制耦合、特征耦合

8. 阅读以下伪代码：

```
模块A(flag, score) {
    if (flag == 1)
        return 计算平均(score);
    else
        return 找最大值(score);
}
```

关于模块A与模块B（被A调用）之间的耦合类型，下列说法正确的是：（  ）

A. 数据耦合，因为flag本质上也是一个数据参数
B. 控制耦合，flag虽以数据形式传递，但其语义是控制信息
C. 特征耦合，因为传递了参数
D. 公共耦合，因为多个分支共享score

关于两种方式的耦合类型及优劣，下列说法正确的是：（  ）

A. 方式1更好，传整个结构体更方便；两者都是数据耦合
B. 方式2更好，是数据耦合；方式1是特征耦合
C. 方式1更好，是数据耦合；方式2是特征耦合
D. 两种方式都是特征耦合，无区别

13. 阅读以下代码片段：

```
if (x > 0) {
    处理正数;
}
goto L;
while (y > 0) {
    处理y;
    L: 收尾;
}
```

关于这段代码为何不符合结构程序设计，下列说法正确的是：（  ）

A. goto从if块跳入while内部，破坏了if块的单入口单出口
B. 代码同时使用了if和while两种控制结构，结构过多导致混乱
C. 使用了goto跳转，破坏了程序整体的单入口单出口原则
D. 没有使用for循环，控制结构不完整
